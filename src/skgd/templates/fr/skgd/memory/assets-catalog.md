## Directives de Sourcing

### Génération IA (PixelLab / Blender MCP)
- Idéal pour : Prototypage, art placeholder, style indie
- Limitations : Cohérence de style, exigences spécifiques
- Conseils : Générer plusieurs variations, choisir la meilleure

### Import Externe
- Formats préférés :
  - Sprites : PNG (avec transparence)
  - Modèles : FBX, GLTF
  - Audio : OGG, WAV
- Convention de nommage : `{catégorie}_{nom}_{variante}`

### Paramètres de Qualité
| Plateforme | Échelle Sprite | Taille Texture | Qualité Audio |
|------------|----------------|----------------|---------------|
| PC | 1.0x | 2048px | Haute |
| Mobile | 0.5x | 1024px | Moyenne |
| Web | 0.75x | 1024px | Moyenne |
